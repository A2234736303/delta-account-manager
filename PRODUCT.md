# Product

## Register

product

## Users

Personal Windows user managing Delta Force game accounts locally. The user is not a professional operator and needs clear, direct controls.

## Product Purpose

Track account credentials, status, notes, rental readiness, and time-based reminders without deploying an online system.

## Brand Personality

Calm, practical, trustworthy.

## Anti-references

Avoid decorative dashboards, marketing-style hero pages, dense technical admin panels, and interfaces that require database knowledge.

## Design Principles

- Keep the account list scannable and editable.
- Make status timing visible everywhere it matters.
- Treat reminders as work queues grouped by current job state.
- Prefer familiar desktop controls over novel interactions.
- Keep local-only data boundaries explicit.

## Accessibility & Inclusion

Use clear contrast, keyboard-friendly controls, visible focus states, reduced-motion-safe transitions, and Chinese labels throughout the core workflow.
